=======================================================================

This package provides all the functionality needed to create a WAF engine based 
on coraza.WAF and OWASP CRS

Git repository:
- https://github.com/corazawaf/coraza
- https://github.com/coreruleset/coreruleset

Additionally, the ruleparser package uses the ModSecurity (SecLang) grammar to 
parse .conf rule files into Go code, compiling all of them into a single binary.
